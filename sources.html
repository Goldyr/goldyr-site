Background noise

Found this CSS in some notes and I really liked how it obscures the page just a little and allows me to hide a lot of bad styling, heres the code:

html::after { content: ""; display: block; position: fixed; left: -32px; top: 0; width: calc(100% + 32px); height: calc(100% + 32px); background-image: url("./scanlines.png"); background-position: 0 0; background-repeat: repeat; z-index: 10000000; animation: ScanlineAnimationRight 0.4s linear infinite; pointer-events: none; opacity: 0.8; } @keyframes ScanlineAnimationLeft { 0% { transform: translateX(0px) translateY(0px); } 100% { transform: translateX(-32px) translateY(-32px); } } @keyframes ScanlineAnimationRight { 0% { transform: translateX(0px) translateY(0px); } 100% { transform: translateX(32px) translateY(-32px); } }
